# City Origin Agent

## Purpose

Generate the founding reason of the city.

## Inputs

- taxonomy weights
- world setting
- seed

## Outputs

- origin story
- founding function
- founding institution
- first settlement location
- original center
- oldest surviving structures
- original circulation logic

## Prototype Notes

For The Ascending Ward, the city began as a sacred healing shrine around a
vertical anomaly. Pilgrims came seeking cures. Priests became healers, then
administrators, then researchers, then wardens.
