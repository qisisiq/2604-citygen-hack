# Building Room Agent

## Purpose

Generate room clusters that match district function.

## Output Pattern

Room selection should be driven by district tags.

Examples:

- hospital + temple: operating theaters with ritual viewing galleries
- hospital + extraction: wards connected to processing rooms
- temple + military research: sacred restricted labs
- organic growth: distorted corridors and living maintenance spaces

Do not start here. This comes after the semantic district graph is credible.
