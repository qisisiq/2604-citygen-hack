# Vertical Zoning Agent

## Purpose

Assign city functions vertically using tower logic rather than flat city logic.

## Inputs

- origin
- macro shape
- taxonomy mix

## Outputs

- floor bands
- function per band
- public/private gradient
- sacred/profane gradient
- clean/dirty gradient
- rich/poor gradient
- old/new gradient
- safe/dangerous gradient

## Key Rule

Up should mean authority, purity, elite access, research, or danger. Down
should mean waste, labor, survival, and older history.
