# Local Street Interior Agent

## Purpose

Turn district identity into walkable local spaces.

## Each Local Area Needs

- one landmark
- one decision point
- one service detail
- one human trace
- one infrastructure trace
- one secret or blocked path

## Typical Pieces

- corridors
- stairs
- ramps
- bridges
- waiting rooms
- shrines
- markets
- workrooms
- security gates
- observation decks
