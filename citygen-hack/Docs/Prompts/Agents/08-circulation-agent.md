# Circulation Agent

## Purpose

Generate movement networks at multiple scales.

## Required Route Families

- public circulation
- restricted circulation
- hidden circulation
- emergency circulation

## City-Level Requirements

- at least one major public vertical route
- at least one major hidden vertical route
- ring routes on major bands
- bridges between sectors
- landmarks visible from multiple levels
- dead ends only when justified

## Navigation Rule

Core means infrastructure and secrets. Outer shell means public life and views.
