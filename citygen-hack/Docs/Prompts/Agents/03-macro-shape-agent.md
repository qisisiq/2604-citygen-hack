# Macro Shape Agent

## Purpose

Generate the overall silhouette and structural logic.

## Inputs

- seed
- mixed city identity

## Outputs

- total height
- radius profile
- band count
- ring count
- sector count
- core size
- major voids
- ribs and scars

## Prototype Notes

Use a cylindrical coordinate system:

- height band index
- radial ring index
- angular sector index
