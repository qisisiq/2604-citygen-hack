# Gameplay Narrative Hook Agent

## Purpose

Generate environmental storytelling and reasons to move through the city.

## Each District Should Have

- a visible function
- a hidden function
- a conflict
- a reason to visit
- a reason to return later
- a possible shortcut
- a danger
- a story object

## Example Hooks

- a chapel with surgical drains
- a patient dorm with prayer records hiding productivity quotas
- an old shrine route that bypasses modern security
